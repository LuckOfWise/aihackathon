# Design Principles — Examples

普遍デザイン原則（[principles.md](./principles.md)）の Good / Bad 具体例。

詳細なコード例は [ui-ux.examples.md](./ui-ux.examples.md) を参照。同ファイルに以下 9 項目の Good / Bad が掲載されている:

1. 中央集中 + モノラベル（SaaS ヒーロー + 3 カード回避）
2. Dense List（カードグリッド回避）
3. アバターピル（独立ログアウトボタン回避）
4. ボタン階層（Primary / Outline / Ghost / Danger）
5. モノラベルブロック（メタ列挙）
6. 日本語タイポグラフィ
7. 戻るナビゲーション（テキスト矢印回避）
8. フォーム（max-width 680px + margin auto + ヒント）
9. インタラクション必須ルール（cursor/transition/focus-visible）

プロジェクト固有の例は [ui-ux.examples.md](./ui-ux.examples.md) 側を直接編集し、普遍パターンとして昇格できる例が見つかったらこのファイルに引き抜く。
