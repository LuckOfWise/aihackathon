# Vue Rules

## Principles

- Composition API (defineComponent + setup関数)
- ローカル状態管理 (reactive() + computed())
- スコープドスタイル (`<style scoped lang="sass">` 必須)
- カスタムイベントはkebab-case (`@on-click-fill`, `@on-change-time-input`)
